# 33 Geniea 动漫表情表 Prompt（4 个复杂度级别）

- **来源**: [Geniea Anime Character Design Prompts](https://www.geniea.com/prompts/gpt-image-anime-character-design)
- **来源类型**: 提示词指南
- **可信度**: 二手来源
- **素材类型**: prompt
- **采集时间**: 2026-05-04

## Prompt

**Beginner — 表情表：**

```
Create a anime image of expression sheet featuring character design. The composition should feel serene with sunset rim light. Use high-contrast monochrome throughout. Every detail should be intentional — from the color transitions to the overall mood. The final result should look like a professional anime piece.
```

**Intermediate — 服装变体：**

```
Create a anime image of costume variation featuring character design. The composition should feel intimate with neon ambient glow. Use analogous warm palette throughout. Every detail should be intentional — from the background atmosphere to the overall mood. The final result should look like a professional anime piece.
```

**Intermediate — 动作姿势：**

```
Create a anime image of action pose featuring character design. The composition should feel dynamic with moonlit scene. Use neon accent colors throughout. Every detail should be intentional — from the material textures to the overall mood. The final result should look like a professional anime piece.
```

**Advanced — 全身转面：**

```
Create a anime image of full turnaround featuring character design. The composition should feel dramatic with golden hour warmth. Use desaturated vintage tones throughout. Every detail should be intentional — from the foreground elements to the overall mood. The final result should look like a professional anime piece.
```

## 备注

Geniea 提供的四个复杂度级别的动漫角色设计 prompt。关键模式：(1) 固定前缀结构（"Create a anime image of [type] featuring character design"）；(2) 每个都包含构图情绪 + 光线 + 色彩系统 + 质量声明；(3) 高级版（full turnaround）使用 "desaturated vintage tones" 和 "golden hour warmth" 创建更强的视觉一致性。注意这些 prompt 没有"上传参考图"依赖——纯粹通过文本描述来定义角色设计表类型。
