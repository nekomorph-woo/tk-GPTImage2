# 38 品牌身份 DNA 提取（文本 DNA 结构参考）

- **来源**: [@Just_sharon7](https://x.com/Just_sharon7)
- **来源类型**: 社区画廊
- **可信度**: 社区来源
- **素材类型**: prompt
- **采集时间**: 2026-05-04

## Prompt

（基于 @Just_sharon7 的品牌身份系统生成。该 prompt 将上传的品牌 Logo/视觉素材提取为完整的品牌 DNA 文档，包含色彩、字体、图形元素、应用规范等。）

```
Analyze the uploaded brand image and create a complete brand identity system including:
- Color palette (extract all brand colors with hex codes)
- Typography recommendations (primary and secondary fonts)
- Logo usage guidelines
- Pattern/texture elements
- Iconography style
- Application examples (business card, social media, packaging)
Maintain exact brand consistency across all elements.
```

## 备注

虽然这是品牌身份而非角色身份，但 "DNA 提取" 的概念完全适用于角色一致性：将角色的视觉元素提取为结构化文档（色彩/发型/服装/配饰/比例），作为后续所有生成的唯一参考源。品牌身份系统的结构（色彩→字体→图形→应用规范）可以直接映射到角色 DNA 系统（肤色→发型→服装→配饰→比例→表情）。
