# 50 WWE 摔角电影化故事板（角色动作一致性）

- **来源**: [@Ankit_patel211](https://x.com/Ankit_patel211)
- **来源类型**: 社区画廊
- **可信度**: 社区来源
- **素材类型**: prompt
- **采集时间**: 2026-05-04

## Prompt

```
Create a cinematic wrestling storyboard featuring [wrestler name/description]. The character has [specific physical features: build, skin tone, hairstyle, facial hair, tattoos] wearing [specific ring attire with colors and design details]. The storyboard shows 6 sequential panels:

Panel 1: Entrance — walking down the ramp with [lighting/effects]
Panel 2: Ring entrance — climbing through ropes, crowd reaction
Panel 3: Face-off — standing across from opponent, intense stare
Panel 4: Signature move — [specific wrestling move description]
Panel 5: Pin attempt — [specific pin description]
Panel 6: Victory celebration — [specific celebration pose]

Maintain exact character consistency across all 6 panels — same face, same body proportions, same ring attire, same skin tone, same tattoos. Cinematic lighting with dramatic shadows and spotlight effects. No text, no watermark. Dark cinematic color grading.
```

## 备注

动作序列中的角色一致性策略：(1) 在 prompt 开头完整定义角色的物理特征（体型/肤色/发型/面部毛发/纹身）和服装（精确到颜色和设计细节）；(2) "Maintain exact character consistency across all 6 panels" + 逐项列出不变量（face/body proportions/ring attire/skin tone/tattoos）；(3) 全局风格锁定（"Cinematic lighting with dramatic shadows" + "Dark cinematic color grading"）。展示了在 6 个不同动作场景中保持角色一致性的文本策略。
