# 04 五元素 Prompt 公式与文本角色一致性策略

- **来源**: [NemoVideo GPT Image 2 Prompts Guide](https://www.nemovideo.com/blog/gpt-image-2-prompts-templates-guide)
- **来源类型**: 博客指南
- **可信度**: 二手来源
- **素材类型**: guide
- **采集时间**: 2026-05-04

## 指导要点

- **5 元素 Prompt 公式**：Subject（主体） + Style（风格） + Lighting（光线） + Details（细节） + Purpose（用途）
- **多面板角色一致性关键**："keep character descriptions identical across prompts"——跨 prompt 逐字复用角色描述，不使用近义词替换
- **Fal.ai 模板结构**：Scene/Subject/Details/Use-case/Constraints 五段式，将一致性约束放在 Constraints 段落中
- **表情关键词精确性**：指定具体表情名称（如 "wide-eyed surprise"、"gentle smile"）比模糊描述（如 "happy expression"）效果更好
- **Pose 描述精确性**：指定身体角度、手臂位置、视线方向等具体参数
- **一致性约束模板**：如 "maintain exact same face and body proportions across all panels"

## 原文摘要

NemoVideo 的 GPT Image 2 prompt 指南提出了五元素公式（主体+风格+光线+细节+用途），并提供了 10 个模板。其中 Comic Panel 模板特别强调多面板间角色一致性的文本实现方法：逐字复用角色描述，不使用近义词替换。文章还引用了 Fal.ai 的五段式模板，建议将一致性约束放在 Constraints 段落中。
