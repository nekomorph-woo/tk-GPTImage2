# 08 Geniea 指南：动漫角色设计 Prompt 最佳实践

- **来源**: [Best Anime Character Design GPT Image Prompts](https://www.geniea.com/prompts/gpt-image-anime-character-design)
- **来源类型**: 提示词指南
- **可信度**: 二手来源
- **素材类型**: guide
- **采集时间**: 2026-05-04

## 指导要点

- **动漫角色设计 Prompt 的分层结构**：
  1. 基础声明（"Create an anime image of [subject] featuring character design"）
  2. 构图情绪（"The composition should feel serene/dynamic/dramatic"）
  3. 光线描述（"with sunset rim light / neon ambient glow / moonlit scene"）
  4. 色彩系统（"Use high-contrast monochrome / analogous warm palette / neon accent colors"）
  5. 质量声明（"professional anime piece"）
- **推荐参数配置**：Quality=HD, Size=1024x1024, Background=opaque
- **保持一致性的技巧**：
  - 从简洁 prompt 开始，逐步添加动漫特定细节
  - 引用具体光线条件给 GPT Image 更清晰的方向
  - 调整 Size 找到质量与速度的最佳平衡点
  - 组合多种风格描述符获得最动态的结果
  - 运行多次迭代并组合不同输出中的元素
- **常见错误**：
  - 跳过风格特定术语——GPT Image 需要特定的动漫词汇才能产出地道结果
  - 塞入太多概念——保持角色设计 prompt 聚焦于统一的美学
  - 不调整 Size——默认值很少给出最佳动漫结果

## 原文摘要

Geniea 的动漫角色设计 prompt 指南提供了分层结构模板，从基础声明到质量声明的五个层次。核心建议是保持 prompt 聚焦于统一的美学，使用动漫特定术语，并从简洁开始逐步迭代。指南还提供了表情表、服装变体、动作姿势、全身转面等四类动漫角色设计 prompt 示例，展示了不同复杂度的文本描述方式。
