## 指导要点

- **日系角色设定页关键词**："professional Japanese character setting page" 可引导输出日系动漫设定风格
- **标准组件列表**：full body turnaround (front/side/back)、consistent proportions、multiple facial expressions、outfit breakdown、close-ups of accessories/shoes/hands/clothing details
- **"consistent proportions"** 是日系角色表中强调最多的关键词
- **16:9 横版**是日系设定页的常用比例
- **Close-up 细节**（配饰、鞋子、手部、服装）是日系角色设定页区别于西方 model sheet 的特色

## 原文摘要

GLBGPT 的 GPT Image 2 精选 prompt 中包含一套日系动漫角色参考表模板。Prompt 以 "Anime character design reference sheet" 开头，指定 16:9 横版布局，要求包含正面/侧面/背面转角度、多种表情、服装分解、配饰/鞋子/手部/服装细节特写。

该模板强调 "professional Japanese character setting page" 这一关键短语，可有效引导模型输出符合日系动漫行业标准的角色设定页风格。close-up 细节部分是日系设定页的特色，包含配饰特写、手部特写等微距视图。
