# 42 对话式穿搭展示 Prompt

- **来源**: [@Just_sharon7](https://x.com/Just_sharon7/status/2050211711450304686)
- **来源类型**: 一手样本
- **可信度**: 一手来源
- **素材类型**: prompt
- **采集时间**: 2026-05-04

## Prompt

```
Okay wait— I need you to see this outfit because I literally cannot stop wearing it… [transition cut] …like are you KIDDING me? The pink, the yellow, the red bag? It just works. This is my 'I'm that girl' outfit and I'm not taking it off."
```

## 备注

对话式 prompt 的范例——用第一人称口语化描述穿搭，配合 [transition cut] 标记视频转场。角色通过语气和态度（自信/夸张）建立人格，而非通过外貌描述。展示了非传统角色建立方式。
