## 指导要点

- **角色参考表核心组件**：多角度全身视图（正面/侧面/背面）、表情集、服装细节分解、配色方案
- **16:9 横版布局**适合展示内容丰富的角色设定表
- **"Production-ready character sheet"** 是有效关键词，暗示专业级别输出
- **3-view turnaround + expressions + costume breakdowns + color palette** 是完整角色表的标准组合
- **奇幻风格角色**需要额外指定种族特征（精灵耳朵、魔法纹路等）作为一致性锚点
- **角色名可作为一致性标记**：在多个面板中重复角色名可强化身份关联

## 原文摘要

Pixverse 的 GPT Image 2 评测文章包含一套奇幻 RPG 女法师角色参考表模板。该模板使用 16:9 横版布局，要求生成 3 视角转角度（正面/侧面/背面）、表情集、服装分解图和配色方案。

Prompt 结构强调 "production-ready character sheet" 关键词以提升输出质量，并详细描述了角色的魔法属性、服装细节（法师长袍、护甲元素、魔法饰品）作为一致性锚点。该模板展示了"具体角色设计 + 标准角色表格式"的组合方式。
