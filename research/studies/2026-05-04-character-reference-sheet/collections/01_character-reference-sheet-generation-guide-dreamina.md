# 01 Dreamina 角色转角度表生成指南与 4 套模板

- **来源**: [Dreamina](https://dreamina.capcut.com/resource/gpt-image-2-for-character-turnaround-sheets)
- **来源类型**: 博客指南
- **可信度**: 二手来源
- **素材类型**: guide
- **采集时间**: 2026-05-04

## 指导要点

- **5 步工作流**：定义角色 → 编写整合 prompt → 配置布局 → 生成 → 细化导出
- **转角度表核心要素**：正面/侧面/背面视图，T-pose 或 A-pose，纯色背景，比例参考线
- **Identity Anchors（身份锚点）**：在 prompt 中使用固定的角色描述词（发色、瞳色、服装、体型等），确保多次生成之间角色一致
- **3:2 横版比例**是转角度表的最佳画幅，留出足够横向空间展示多角度视图
- **一致性关键词**：使用 "consistent proportions"、"exact same face/body proportions"、"identical styling" 等短语强化跨面板一致性
- **表情面板**可在转角度表中一并生成，将表情描述作为附加面板指令
- **品牌吉祥物**使用 A-pose（自然站姿），角色使用 T-pose（双臂平展），便于展示轮廓

## 原文摘要

Dreamina 提供了完整的 GPT Image 2 角色转角度表生成教程，包含 4 套可直接使用的 prompt 模板：

1. **男主制作角色表**：3:2 横版，正面/侧面/背面，纯白影棚背景，电影级灯光
2. **女性盗贼游戏角色转角度**：T-pose，身份锚点关键词，写实光照
3. **角色包（转角度 + 6 表情头像）**：少年主角，3:2 横版，表情名称列表
4. **品牌吉祥物转角度**：A-pose，标注视图标签，营销交付，3:2 横版

核心方法论是"先建立角色描述锚点，再用一致性关键词确保面板间角色不变"。建议使用纯色背景以便后期抠图，并推荐 3:2 画幅作为最佳比例。
