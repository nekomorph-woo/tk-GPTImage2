# 45 角色包（转角度 + 6 表情头像）— Dreamina 模板

- **来源**: [Dreamina](https://dreamina.capcut.com/resource/gpt-image-2-for-character-turnaround-sheets)
- **来源类型**: 博客示例
- **可信度**: 二手来源
- **素材类型**: prompt
- **采集时间**: 2026-05-04

## Prompt

```
A character pack featuring a teen protagonist, 3:2 horizontal layout, clean light gray background. Top section: full body turnaround showing front, side, and back views of the character standing in a natural A-pose. Bottom section: 6 expression headshots in a row showing the following emotions: happy, angry, sad, surprised, determined, curious. Character: Japanese teenage boy, 16 years old, messy dark brown hair, bright amber eyes, fair skin with light freckles. Wearing a school uniform with an open jacket, rolled-up sleeves, sneakers. Keep the character fully consistent across all views and expressions — same face, same hair, same proportions. Anime illustration style, clean lineart, vibrant colors, professional character design sheet quality.
```

## 备注

Dreamina 教程中的第 3 套模板——角色包（转角度 + 表情头像组合）。关键点：A-pose（自然站姿，非 T-pose）适合非游戏角色；6 个表情名称明确列出；"Keep the character fully consistent across all views and expressions" 是核心一致性指令。
