# 05 GPT Image 2 多参考图 API 能力与角色一致性应用

- **来源**: [Fal.ai GPT Image 2 Prompting Guide](https://fal.ai/learn/tools/prompting-gpt-image-2) + [NoteGPT GPT Image 2 Guide](https://notegpt.io/blog/gpt-image-2-prompt-guide-use-cases)
- **来源类型**: 博客指南
- **可信度**: 二手来源
- **素材类型**: guide
- **采集时间**: 2026-05-04

## 指导要点

- **gpt-image-2 支持多达 16 张参考图输入**：API 的 edit 端点接受最多 16 张参考图片（file ID 或 URL），远超许多用户的认知。
- **ChatGPT 界面支持最多 8 张图片上传**：NoteGPT 验证了 ChatGPT 网页版一次可以上传 8 张图片。
- **多参考图的角色一致性应用模式**：
  - 上传同一角色的多角度照片（正面、侧面、背面），让模型综合理解角色外貌
  - 上传角色参考表 + 目标场景参考，实现角色在新场景中的渲染
  - 上传多张角色图作为风格参考，保持画风一致
- **角色设计表作为参考图的使用方式**：先生成完整的角色设计表（character sheet），再将该表作为后续所有生成的唯一参考图，这是社区广泛采用的工作流。
- **多轮编辑的链式工作流**：生成→选最佳→编辑→选最佳→编辑...，每轮都将上一轮的最佳输出作为下一轮的参考输入。
- **Fresh Chat 策略**：社区发现每次开始新对话（而非在同一长对话中累积），配合参考图上传，可以有效避免上下文漂移。
- **编辑端点 vs 生成端点**：编辑端点（edit）比生成端点（generate）更适合角色一致性场景，因为编辑端点会显式参考输入图片。

## 原文摘要

Fal.ai 的技术文档确认 gpt-image-2 的 edit 端点支持多达 16 张参考图输入，NoteGPT 则验证了 ChatGPT 网页界面支持 8 张图片上传。这一多参考图能力为角色一致性打开了多种应用模式：多角度参考综合、设计表+场景组合、多角色风格对齐等。社区总结的最佳实践包括：先生成角色设计表，再将其作为后续所有编辑的唯一参考；使用"Fresh Chat"策略避免长上下文漂移；以及链式编辑工作流（每轮输出作为下轮输入）。
