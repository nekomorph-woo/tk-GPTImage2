# 18 从 Logo DNA 构建完整品牌识别系统

- **可信度**: 社区来源

- **采集时间**: 2026-05-04

## Prompt

```json
{
  "prompt": {
    "title": "Agency-Grade Brand Identity System Poster",
    "trigger": "Upload a logo. From it, construct a complete, investment-worthy brand identity system poster — the kind that closes client deals and dominates Behance front pages.",

    "prime_directive": {
      "rule": "Every element — color, tone, shape, texture, personality — must be extracted directly from the uploaded logo.",
      "enforcement": "Nothing generic. Nothing templated. Nothing borrowed. Strip the logo. Decode it. Build an entire visual universe from its DNA."
    },

    "format": {
      "orientation": "Vertical",
      "aspect_ratio": "4:5",
      "layout": "Multi-column grid",
      "composition": "Layered, dense, intentional — zero wasted space"
    },

    "sections": {
      "01_brand_header": "Brand name in commanding typography, 6-word brand statement, three soul descriptors",
      "02_color_system": "3–5 primary colors from logo, 3–5 secondary, accent hits. Per color: swatch + HEX + role label. Gradient blends, color pairings, light/dark mode.",
      "03_typography_system": "Headline (bold), subheadline (structured), body (readable). Hierarchy undeniable at a glance.",
      "04_visual_language": "Image style, lighting quality, texture references. 3–5 art-directed mood board tiles.",
      "05_brand_applications": "Mockups: Product Packaging, Website Hero, Mobile App Screen, Social Media Posts (square/story/banner), Business Card (front/back), Out-of-Home Ad (billboard/transit).",
      "06_design_system": "Buttons (default/hover/disabled), Cards, Input fields, Navigation bar, Spacing scale. Must resemble real design system handoff.",
      "07_iconography": "6–10 icons, same visual grammar as logo. Uniform stroke weight or fill logic."
    }
  }
}
```

## 备注

这个 prompt 展示了"参考图驱动的视觉一致性"在品牌设计领域的应用。"prime_directive" 部分明确要求"从 Logo 中提取每一个视觉元素，构建整个视觉宇宙"——这与角色一致性中的"DNA Template"理念完全一致。JSON 格式的结构化 prompt 使每个板块的定义清晰且可维护。该 prompt 的核心洞察是：一个参考图（Logo）足以驱动整个视觉系统的生成，前提是显式声明"所有元素必须从参考图中提取"。
