# prompts — 共用提示词资源

存放所有技能共享的提示词框架、约束、评分标准和可复用卡片。

## 文件说明

| 文件 | 用途 | 使用方 |
|------|------|--------|
| `PROMPT_FRAMEWORK.md` | 标准提示词结构（前缀、主体、风格、参数） | generate-image, image-research |
| `NEGATIVE_CONSTRAINTS.md` | 失败规避短语清单 | generate-image, image-research |
| `REVIEW_RUBRIC.md` | 图片评分标准 | image-research |
| `CHATGPT_MODE.md` | ChatGPT Plus 手动生图模式说明 | generate-image |
| `prompt-cards/` | 可复用提示词卡片（如 `cinematic-still.md`、`product-poster.md`） | generate-image, image-research |
