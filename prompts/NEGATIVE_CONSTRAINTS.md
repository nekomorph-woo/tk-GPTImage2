# 负面约束

这些约束要克制使用。只写当前图片最需要规避的失败点。

## 使用原则

1. **克制选取**：每次只用 3-5 条最相关的约束，不堆砌全部
2. **长度控制**：负面约束占 prompt 总长度的 ~15-20%，低于 200 字符开始失去效果
3. **正面锚定**：优先用正面描述（"柔和自然光"）替代负面禁止（"不要不自然的光"），负面约束只用于正面描述无法覆盖的情况
4. **按场景选取**：参考流派匹配的负面约束字段，不同流派自动注入对应类别

---

## 通用

```text
无水印，无虚假 Logo，无额外文字，无重复主体，无扭曲解剖，背景不要杂乱，
不要颗粒噪声，不要 JPEG 压缩痕迹，不要过度锐化，不要低分辨率模糊
```

## 人物

```text
手部自然，手指连贯，面部自然，真实皮肤纹理，不要塑料皮肤，不要多余肢体，
眼睛对称且目光一致，牙齿自然不发光，头发不融入背景
```

## 产品

```text
只出现一个产品，保留准确轮廓，不要漂浮零件，不要虚假品牌 Logo，不要指纹，不要弯曲边缘，
不要几何变形，不要不合逻辑的反射
```

## 字体排版

```text
只出现指定文字，不要额外随机文字，不要拼写错误，字体可读，高对比度，间距干净，
不要字母间距异常，不要文字模糊或重叠
```

## 中文文字

```text
只出现指定中文标题，不要额外中文字符，不要英文文字，中文字体清晰可读
```

## 编辑图

```text
保留原始构图，保留主体身份，不要改变产品比例，只修改指定区域，
不要引入原图不存在的新元素
```

## 技术质量

```text
不要深度伪影，不要莫尔条纹，不要过度 HDR，不要色晕（chromatic aberration），不要镜头光晕过强
```

## 解剖细节

```text
脖子比例正常，肩膀对称，四肢长度一致，关节弯曲方向正确，手指数量正确
```

## 风格控制

```text
不要混合多种画风，不要漫画与写实混合，不要 3D 与 2D 混合，风格保持一致
```

## 光影与色彩

```text
阴影方向一致，不要矛盾光源，不要不自然的彩色阴影，色彩不过饱和，不要色调分离
```
