# 负面约束

这些约束要克制使用。只写当前图片最需要规避的失败点。

## 通用

```text
无水印，无虚假 Logo，无额外文字，无重复主体，无扭曲解剖，背景不要杂乱
```

## 人物

```text
手部自然，手指连贯，面部自然，真实皮肤纹理，不要塑料皮肤，不要多余肢体
```

## 产品

```text
只出现一个产品，保留准确轮廓，不要漂浮零件，不要虚假品牌 Logo，不要指纹，不要弯曲边缘
```

## 字体排版

```text
只出现指定文字，不要额外随机文字，不要拼写错误，字体可读，高对比度，间距干净
```

## 中文文字

```text
只出现指定中文标题，不要额外中文字符，不要英文文字，中文字体清晰可读
```

## 编辑图

```text
保留原始构图，保留主体身份，不要改变产品比例，只修改指定区域
```
