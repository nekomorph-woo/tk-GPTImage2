## Prompt

A hand-drawn crayon illustration of an Instagram profile page based on the uploaded image. The entire UI is recreated in a playful, childlike crayon art style with visible pencil strokes, soft uneven coloring, and textured paper background. The profile picture is illustrated as a cartoon version with simple facial features and bold outlines. Interface elements like username, follower count, buttons (Follow, Message), and icons are sketched with imperfect lines and bright crayon colors.
Surround the profile with doodles like stars, flowers, rockets, smiley faces, and small playful shapes scattered across the background. The feed grid below shows mini crayon-style thumbnails representing posts, each simplified into colorful sketches with bold outlines. Use a warm, vibrant color palette with soft shading and a slightly grainy paper texture to enhance the handmade feel.
Keep the layout faithful to Instagram UI but stylized as if drawn by hand with crayons, slightly imperfect, cozy, and whimsical.
