[BRAND = TOYOTA]

Act as a Senior Automotive Art Director blending tactical GR-style UI density with cinematic expedition luxury, creating a collector-grade Land Cruiser poster with controlled aggression and heritage dominance.

PHASE 1: CONCEPTUAL FRAMEWORK

Create a cinematic + tactical hybrid poster where legacy meets battlefield intelligence.

This is not just a legend.
This is a system built to conquer terrain.

Core Idea:
"LEGEND WITH INTELLIGENCE"

Aesthetic:
- Land Cruiser cinematic heritage
- Fortuner GR tactical UI layering
- Military precision meets expedition luxury
- Controlled chaos (structured, not messy)

Emotion:
Authority.
Control.
Unstoppable intelligence.

PHASE 2: HERO VEHICLE (DOMINANT PRESENCE)

- Vehicle: Toyota Land Cruiser (latest gen)
- Angle: Low-angle 3/4 front (slightly aggressive tilt)
- Placement: Center dominant, slightly breaking lower frame
- Details: Subtle dust (premium rugged, not messy), Strong reflections on body, Massive wheels + terrain grip, Warm sunlight highlights

PHASE 3: TYPOGRAPHY (HYBRID STYLE)

- Massive background text: LAND CRUISER
- Style: Bold, wide, slightly textured (sand/grain feel)
- Text sits behind vehicle, slight opacity blend into environment, vehicle overlaps text
- Secondary: GR-style accent text: "LEGEND MODE" "ALL TERRAIN LOCKED"

PHASE 4: MULTI-SCENE LAYERING

Combine cinematic + tactical panels:
1. HERO VEHICLE (main)
2. DESERT ACTION (sand drift)
3. MOUNTAIN CLIMB (extreme angle)
4. DETAIL SHOT (grille / wheel)
5. INTERIOR (luxury cockpit)

PHASE 5: GRAPHIC SYSTEM (GR TACTICAL UI)

Add 10–14 elements:
- Crosshair targeting UI
- "4X4 MODE LOCKED"
- Terrain selector icons: ROCK / SAND / MUD / SNOW / SLOPE
- Torque indicator: "TORQUE READY"
- Coordinate system: LAT / LONG / ELEVATION
- Barcode + serial strips
- Direction arrows
- Mechanical diagram (drivetrain layout)
- Grid overlays
- Warning labels

Style: Structured, sharp, aggressive UI (NOT random)

PHASE 6: COLOR SYSTEM

- Base: Sand beige + desert tones
- Contrast: Deep black shadows
- Accent: GR RED (used sparingly in UI + highlights)
- Lighting: Golden hour + cinematic contrast

PHASE 7: PANEL SYSTEM (TACTICAL STRIP)

Bottom section: 4 panels (wheel traction, interior cockpit, side profile, off-road action)
Each panel: Hard borders, slight glow or edge highlight, high contrast imagery

PHASE 8: MICRO TYPOGRAPHY (DENSE SYSTEM)

- "ALL TERRAIN CONTROL" / "LEGENDARY RELIABILITY" / "BUILT FOR EXTREMES" / "4X4 LOCKED"
- Serial numbers / codes
Placement: Corners, overlays, panels

PHASE 9: COMPOSITION

Hierarchy: 1. Vehicle 2. Typography 3. UI overlays 4. Panels
Flow: Diagonal forward energy + grounded base

PHASE 10: DEPTH & LAYERING

- Foreground: Dust particles + UI overlays
- Mid: Vehicle + typography interaction
- Background: Desert + mountain + topo maps
Add atmospheric haze

PHASE 11: BRAND INTELLIGENCE

Land Cruiser + GR fusion: Heritage + endurance, Intelligence + control, Luxury + off-road dominance
Tone: Authoritative. Engineered. Unstoppable.

TECHNICAL SPECS:
- 4:5 vertical
- Ultra high resolution
- Cinematic color grade + UI sharpness
- Balanced grain (10–15%)

## 用途

生成汽车品牌战术风格信息图海报，11 阶段设计流程（概念框架→英雄车辆→混合排版→多场景分层→GR 战术 UI 图形系统→色彩系统→面板系统→微排版→构图→景深分层→品牌适配），以丰田兰德酷路泽为例，4:5 竖版，军事精确感+探险奢华风格，品牌/车型/颜色可替换模板。
