## Prompt

```
基于我提供的手掌照片，生成一份完整的「手相解析指南」。

要求如下：

* 核心内容：对手掌进行详细分析，包括手型、掌丘、三大主线（生命线、智慧线、感情线）以及命运线等，给出清晰、有逻辑的解读
* 视觉风格：整体设计为高端、极简风格（类似高端杂志/咨询报告）
    * 纯白背景
    * 细线条（thin line）设计
    * 圆角卡片（rounded cards）分区排版
    * 留白充足，排版干净克制
    * 视觉质感要"高级、有设计感"
* 版式结构建议：
    1. 标题区（如：你的手相解析指南）
    2. 整体手型分析
    3. 三大主线解析
    4. 其他重要线纹（命运线、太阳线等）
    5. 综合解读（性格 / 事业 / 感情 / 财运 / 健康）
* 重点要求：
    * 根据照片中的手掌特征进行合理分析（尽量贴合实际纹路）
    * 生成一个黑白极简线稿的小插图：
        * 仅用黑线在白底上勾勒手掌
        * 清晰标出三大主线位置
        * 风格简洁、像设计图标/说明图
* 语言要求：
    * 最终输出全部使用中文
    * 文案风格：简洁、有质感、略带高级感（类似品牌白皮书或咨询报告）
```

## 用途

基于手掌照片生成中文手相解析信息图，高端极简风格（纯白背景+细线条+圆角卡片分区），包含手型分析、三大主线解析、综合解读（性格/事业/感情/财运/健康），附带黑白线稿手掌标注插图，中文品牌白皮书质感。
