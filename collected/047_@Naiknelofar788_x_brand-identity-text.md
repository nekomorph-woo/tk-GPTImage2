## Prompt

```
Using the uploaded logo, generate a highly detailed, premium brand identity system poster.

GOAL:
Create a complete, visually rich brand kit that looks like it was made by a top design agency.

CORE RULE:
Everything must be derived from the uploaded logo: colors, style, tone, personality. No generic outputs.

STRUCTURE:
Vertical 4:5 poster. Multi-layered grid system. Dense but clean composition.

TOP SECTION:
- Brand name (clean typography)
- Short brand statement (max 6 words)
- 3-word brand identity (e.g. "Modern / Bold / Minimal")

COLOR SYSTEM:
- Primary palette (3–5 colors), Secondary palette (3–5 colors), Accent colors
- Per color: large blocks, HEX codes, usage indicators
- Gradient examples, color combinations

TYPOGRAPHY SYSTEM:
- Headline, subheadline, body text styles with real text examples
- Hierarchy clearly visible

VISUAL LANGUAGE:
- Image style, lighting direction, texture/material inspiration
- 3–5 visual tiles (image-style previews)

BRAND APPLICATIONS:
- Product packaging, website hero section, mobile UI screen
- Social media posts (3 variations), business card, billboard/ad
- Each must feel consistent with the brand

LAYOUT SYSTEM:
- UI blocks, card components, spacing system
- Buttons, cards, layout examples

ICONOGRAPHY:
- 6–10 icons in brand style, consistent line/fill style

PATTERNS & ELEMENTS:
- Background patterns, decorative shapes, visual motifs derived from logo

MICRO DETAILS:
- Shadows, material textures, reflections, depth layers

VISUAL STYLE:
- Modern editorial + tech design hybrid
- Extremely clean but information-rich
- Layered composition, strong hierarchy
- 30–50 visual elements total, mix of large + small

IMPORTANT RULES:
- No empty space, no generic placeholders
- Everything must feel intentional, all elements must visually connect

FINAL FEEL:
Like a Behance top project, a real agency brand guideline board, something clients would pay for.
NOT basic, minimal, or template-like.
```

## 用途

上传 logo 生成完整专业品牌识别系统海报（4:5 竖版），从 logo 提取全部视觉 DNA，覆盖品牌头部→色彩系统→排版→视觉语言→品牌应用 mockup（包装/网站/APP/社媒/名片/户外广告）→布局系统→图标→图案元素→微细节（阴影/纹理/反射），Behance 级质量，30-50 视觉元素密集布局。
