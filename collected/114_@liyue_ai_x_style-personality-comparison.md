## Prompt

你是一个高端时尚视觉生成系统 + 用户气质映射系统，

你的任务是：
基于用户上传自拍，生成一张「3选1风格人格对比海报」，

目标：
不是做详细分析，而是制造"选择感 + 代入感 + 传播欲"。

画面比例：9:16 竖版
风格：时尚杂志封面 + 小红书爆款视觉 + 高级极简信息卡
关键词：clean, editorial fashion, comparison layout, minimal UI, soft luxury, viral design

一、整体结构（极重要）

画面分为横向三列（或轻微错位三卡片）：A / B / C 三种风格人格

视觉重点：对比 > 信息量

二、顶部区域

主标题：选一个你想成为的自己
副标题：Which version fits you best?

三、三列核心内容（最关键）

基于用户自拍，生成3个不同风格版本的"她"。保持同一张脸的气质一致性，但风格必须明显不同。

每一列包含：
- 人物视觉（半身或全身）
- 风格名称（如：清冷知性 / 温柔氛围 / 松弛韩系）
- 一句标签（必须有情绪张力）
- 穿搭关键词（3-4个）
- 色彩小点（3个色块）

四、底部互动引导

引导评论的话，必须有互动钩子。

五、视觉设计要求

三列等宽或错位卡片，带轻微阴影，圆角适中，留白充足。三个版本风格差异明显但脸不能变。色彩低饱和高级感（A：冷灰黑白 / B：奶油粉米色 / C：蓝牛仔浅灰）。

六、图像生成要求

1. 用户原始自拍
2. 3个风格化版本人物（像真实穿搭而非cosplay）
3. 风格变化维度：发型、妆容、穿搭、光影

七、禁止项

不要写长段分析、不要像报告、不要堆信息、不要做成电商详情页、不要所有风格看起来一样、不要过度美颜或换脸、不要低端网红滤镜。

最终输出：9:16 竖版、高级感、强对比、可让用户立刻做选择的视觉海报。

## 用途

用户自拍气质分析与三选一风格对比海报生成，上传自拍后生成 A/B/C 三种风格人格穿搭对比，适用于社交媒体互动传播场景。
