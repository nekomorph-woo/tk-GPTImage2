# 提示词套路

## 电影剧照

公式：

`具体人物 + 具体地点 + 镜头/裁切 + 真实光源 + 触感细节 + 克制调色 + 无文字`

诊断：

- 如果画面普通，增加具体地点和三个真实物件。
- 如果画面太假，减少风格堆叠，并指定真实光源。
- 如果主体弱，明确裁切、大小和位置。

## 高级产品海报

公式：

`单一产品 + 受控表面 + 反射规则 + 留白 + 短精确标题 + Logo 限制`

诊断：

- 如果产品重复，写明“只出现一个产品”。
- 如果显得廉价，减少颜色并指定反射行为。
- 如果文字失败，缩短文字并加强排版约束。

## 中文编辑海报

公式：

`短标题 + 标题位置 + 诗意场景物件 + 印刷/编辑质感 + 严格无额外文字规则`

诊断：

- 如果中文乱码，缩短中文，并让它成为唯一文字。
- 如果布局混乱，指定标题区域和主体区域。
- 如果氛围弱，增加天气、材质和反射光细节。

## 一致性角色

公式：

`名字 + 年龄范围 + 脸部标记 + 服装 + 1-2 个配饰 + 中性参考姿势 + 纯背景`

诊断：

- 如果身份漂移，先创建参考设定图，再基于该图编辑。
- 如果服装漂移，列出准确服装，并禁止额外配饰。
- 如果场景过于复杂，回到摄影棚背景。

## 触感 3D 图标

公式：

`单一物体 + 材质组合 + 等距视角 + 棚拍光 + 干净边距 + 无文字`

诊断：

- 如果图标混乱，减少到一个物体和一个强调色。
- 如果画面太平，指定半透明、圆角、光泽或哑光等材质行为。
- 如果变成 Logo，禁止字母和符号。

## 自主研究循环

当用户要求持续改进实验室时：

1. 搜索当前官方文档和公开社区案例。
2. 总结类别和提示词结构，不长段复制第三方文字。
3. 在 `studies/` 下新增一份简洁研究笔记。
4. 在 `recipes/recipes.json` 中新增或改进一个 recipe。
5. 只有当提示词卡片代表可复用模式时，才新增到 `prompts/prompt-cards/`。
6. 运行 `npm run doctor` 和 `npm run recipes`。
