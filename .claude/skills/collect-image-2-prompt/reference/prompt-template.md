# 收集 Prompt 记录模板

保存路径：`collected/`

文件名格式：`<编号>_<作者>@<平台>_<关键词>.md`，例如 `001_@jsaborido_x_cinematic.md`

编号从 `collected/` 现有文件中自动递增（取最大编号 +1）。

---

## 模板

```markdown
# #<编号>

- **来源**: [@作者用户名](推文/评论 URL)
- **采集时间**: YYYY-MM-DD
- **Prompt 位置**: 主推文 / 博主评论 / 反推
- **搜索词**: <本次收集使用的搜索词> / 反推
- **组**: <可选，同源多图时填首图编号，如 `068`；单图时省略此字段>

## Prompt

<提取到的 prompt 内容>

## 用途

<一句话描述该 prompt 生成什么效果的图片，≤150字>

## 标签

<tag1, tag2, tag3>

## 图片样例

- ![样例 1](图片 URL)
- ![样例 2](图片 URL)

> 如无图片则删除此节

## 原始文本

<推文/评论完整原文，保留换行和格式>
```
